# openWithChrome

To open files from the **terminal** in Unix w/ **Google Chrome:**

##Using Bash:

1. Open terminal
2. cd .bash_profile
3. Add the following line to the end:

-> alias chrome='open -a "Google Chrome"' <-

##Using Zsh:

1. Open terminal
2. cd .zshrc
3. Add the following line to the end:

-> alias chrome='open -a "Google Chrome"' <-

##Test

Open your terminal and type **chrome** continued by *file name*.

###Example:

**chrome index.html**
